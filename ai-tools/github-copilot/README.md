# Mastering GitHub Copilot — Tutorial Series

A comprehensive tutorial series for senior backend developers working with Java/Spring Boot, MySQL, Kafka, Docker, Kubernetes, Azure, Terraform, and GitHub Actions.

---

## Learning Path

Complete these tutorials in order. Each builds on the previous one.

| # | Tutorial | Description | Time |
|---|----------|-------------|------|
| 1 | [Setup & Configuration](01-copilot-setup-and-configuration.md) | Plans, installation across IDEs and CLI, configuration, proxy setup, project-level instructions | 30 min |
| 2 | [Code Completion Mastery](02-code-completion-mastery.md) | Inline suggestions, keyboard shortcuts, partial acceptance, context awareness, multi-line completions | 60 min |
| 3 | [Copilot Chat & Edits](03-copilot-chat-and-edits.md) | Chat panel, participants, slash commands, context references, Edits mode, inline chat, vision | 60 min |
| 4 | [Copilot CLI](04-copilot-cli.md) | `gh copilot suggest`, `gh copilot explain`, shell integration, practical DevOps examples | 30 min |
| 5 | [Copilot for Pull Requests](05-copilot-for-pull-requests.md) | PR summaries, automated code review, Copilot Workspace, custom review instructions | 45 min |
| 6 | [Advanced Patterns & Tips](06-advanced-patterns-and-tips.md) | Custom instructions deep dive, prompt engineering, TDD, extensions, model selection, security | 60 min |
| 7 | [Daily Workflow Playbook](07-daily-workflow-playbook.md) | Practical playbook for using Copilot throughout your entire workday, shortcuts cheat sheet, comparison with Claude Code | 45 min |

**Total estimated time: ~5.5 hours**

---

## Prerequisites

- GitHub account with Copilot subscription (Individual, Business, or Enterprise)
- One of: VS Code, IntelliJ IDEA, or Neovim
- GitHub CLI (`gh`) installed
- Familiarity with Java/Spring Boot, Docker, Kubernetes, Terraform

## How to Use This Series

- **New to Copilot?** Start with Tutorial 1 and go in order.
- **Already using Copilot?** Jump to Tutorial 3 (Chat & Edits) or Tutorial 6 (Advanced Patterns).
- **Want a daily reference?** Bookmark Tutorial 7 (Daily Workflow Playbook).
- **DevOps focused?** Tutorials 2 and 4 have the most Docker/K8s/Terraform examples.

## Key Shortcuts Reference (VS Code)

| Action | Shortcut |
|--------|----------|
| Accept suggestion | `Tab` |
| Dismiss suggestion | `Esc` |
| Next suggestion | `Alt + ]` |
| Previous suggestion | `Alt + [` |
| Accept next word | `Ctrl + Right Arrow` |
| Open completions panel | `Ctrl + Enter` |
| Open inline chat | `Ctrl + I` |
| Open chat panel | `Ctrl + Alt + I` |
| Toggle Copilot | Click status bar icon |
